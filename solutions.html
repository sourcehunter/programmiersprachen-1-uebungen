								let horizontalAnchor = 0.1;
								let verticalAnchor = 0.9;

Exercise 4: Working with Variables

There are an infinite number of combinations, below are a few examples. The smallest number of operations to define all three variables is 5. 1 for d and f and 3 for e.

						
							let d = a * b;
							let d = b + b;
							let d = a + a * a * a;
							
							let e = d + d + d + a + b;
							let e = c * a * a + a / a;
							let e = b * c - a * a * a;
							let e = b * c - d + a;
							let e = (a + b) * b + a;
							let e = (c + b) * a + c;
							
							let f = b / d;
							let f = -(c / a) + b;
							let f = a / a / a;
							let f = b / (a * b);
